This project is a browser-based brick breaker game inspired by classic arcade gameplay.
It is being developed collaboratively as part of a sprint-based workflow using HTML, CSS, and JavaScript.
Players control a paddle to bounce a ball and break bricks while managing limited lives.

Features:
- paddle and ball movement
- brick collision system
- life tracking system
- keyboard, mouse, and button controls

Controls:
- spacebar press / click to launch the ball
- mouse / arrow keys to move the paddle
- esc button press / button click to pause

How to run:
1. clone the repository -- git clone https://github.com/SmallFox16/PowerPuffGirls.git
2. navigate to the project folder -- cd PowerPuffGirls
3. open index.html file in your browser
